How to run
----------
	Windows Machine:
	---------------
	1. open anaconda prompt in windows.
		Note:
			I have executed and tested in below fedora machine.
			Windows - 8 

	2. Give below command 
		python bikeshare_2.py
		
	3. Follow the instructions by giving input by "bikeshare_2.py"

	Fedora-26 Machine:
	-----------------
	1. Install python3,numpy,panda stable version in your Fedora Machine.
		For installation panda refer below link.
		https://pandas.pydata.org/pandas-docs/stable/install.html

	2. Open terminal. 
		Note:
			I have executed and tested in below fedora machine.
			Version: Fedora 26
			Processor: Intel® Core™ i5-3470 CPU @ 3.20GHz × 4 
			OSType: 64bit
			GNOME: Version 3.24.2
			
	3. Provide command as below.
		python3 bikeshare_2.py
		
	4. Follow the instructions by giving input by "bikeshare_2.py"
