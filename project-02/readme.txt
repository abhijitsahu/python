-----------	
1. Open Anaconda Command Prompt (Refer Image: CommadsAnaconda.jpg )
2. GIve below command:
	python E:\SharedFolderWithVM\udacity\final_project_2\project-2\flask_project2.py

	* Environment: production

	  Use a production WSGI server instead.
	* Debug mode: on
	* Restarting with stat
	* Debugger is active!
	* Debugger PIN: 174-395-104
	* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)
4. Then you can open your browser (Ex: Chrome) and look for above IP  which you probably got from above steps i.e http://127.0.0.1:5000/
