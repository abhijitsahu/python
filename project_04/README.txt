Project Name: Movie Trailer Website
===================================

How to run:
-----------
Step-1: Install python 2.7 and install webbrowser module.
Step-2: Open "enetertainment_centre.py" file in python IDLE 2.7.
Step-3: Press F5 , then you would see something image like "project_screen_01.png".
Step-5: Click on any one movie trailler you wish to watch(You can refer "project_screen_*.png" as well). Enjoy!
